Todo lo que necesitas para empezar. Sin conocimientos tecnicos.

Por que existe esta herramienta?

Cuando buscas empleo, normalmente haces esto: abres LinkedIn, buscas ofertas una por una, copias tu CV, lo envias tal cual a cada empresa... y repites. El problema es que un CV generico casi nunca recibe respuesta.

JobHunter automatiza ese proceso. Busca ofertas en LinkedIn, identifica cuales son reales y tienen email del reclutador, y para cada una genera un CV adaptado a lo que esa oferta pide. Despues escribe un email corto y directo, y lo envia. Todo desde tu computadora, con tus datos, sin intermediarios.

No es un hack ni hace nada ilegal. Simplemente navega LinkedIn como lo harias tu, pero mas rapido. Y usa inteligencia artificial (Google Gemini, gratuita) para adaptar tu experiencia real al lenguaje de cada oferta. No inventa nada.

Antes de empezar

Necesitas 3 programas instalados. Son gratuitos y seguros. Si ya los tienes, salta al paso 1.

Python 3.10 o superior

Es el lenguaje en que esta hecho JobHunter. Verificalo escribiendo en la terminal:

python --version

Si no lo tienes: python.org

Google Chrome o Microsoft Edge

JobHunter usa el navegador para acceder a LinkedIn. Solo necesitas tenerlo instalado, el programa lo abre solo.

Git

Para descargar y actualizar JobHunter. Verificalo con:

git --version

Si no lo tienes: git-scm.com

Obtener clave de Gemini (gratis)

JobHunter usa la IA de Google (Gemini) para analizar ofertas y generar CVs. La clave es gratuita.

Como obtenerla

  1. Ve a aistudio.google.com/apikey
  2. Inicia sesion con tu cuenta de Google
  3. Haz clic en "Create API Key"
  4. Selecciona cualquier proyecto (o crea uno nuevo)
  5. Copia la clave. Empieza con AIza...
i
La clave es gratuita. No necesitas tarjeta de credito.
!
No compartas tu clave con nadie. Es personal y esta asociada a tu cuenta de Google.

Configurar Gmail para envio

JobHunter envia los emails desde tu cuenta de Gmail. Necesitas una "contrasena de aplicacion" (no tu contrasena normal).

!
Requisito: tu Gmail debe tener verificacion en dos pasos activada. Si no la tienes, activala en este enlace.

Crear contrasena de aplicacion

  1. Ve a myaccount.google.com/apppasswords
  2. En "Nombre de la app" escribe JobHunter
  3. Haz clic en "Crear"
  4. Copia la contrasena de 16 caracteres que aparece
!
Esta contrasena solo se muestra una vez. Si la pierdes, crea una nueva.

Portfolio y LinkedIn. Ambos opcionales, presiona Enter para saltar.

Tipos de empleo separados por coma. Ejemplo: Backend Developer, Full Stack, Data Engineer

Paso 3 — Preferencias

En una sola pantalla configuras:

Buscar ofertas y aplicar

Todo configurado. Ahora ejecuta:

jobhunter run

Que hace el programa

  1. Busca publicaciones en LinkedIn con tus terminos
  2. Filtra solo ofertas reales con email del reclutador
  3. Te muestra una tabla con las ofertas encontradas
  4. Tu eliges a cuales aplicar (o escribes all para todas)
  5. Genera un CV personalizado para cada oferta
  6. Escribe un email de aplicacion
  7. Envia todo al reclutador

Opciones utiles

  • jobhunter run --time week — ofertas de esta semana
  • jobhunter run --time month — ofertas del mes
  • jobhunter run --auto — aplicar a todas sin preguntar
  • jobhunter --test tu@email.com — modo prueba (te envia a ti)
i
La primera vez usa jobhunter --test tu@email.com para ver como quedan los CVs y emails antes de enviar a reclutadores reales.

Preguntas frecuentes

Dudas comunes al usar JobHunter.

Es gratis?

Si. JobHunter es open source y gratuito. La API de Gemini tiene un plan gratuito suficiente para uso normal.

Es seguro? Mis datos estan protegidos?

Todo se ejecuta en tu computadora. Tus datos nunca se suben a ningun servidor. El unico servicio externo es la IA de Google para analizar ofertas.

Que pasa si mi sesion de LinkedIn expira?

Ejecuta jobhunter login para volver a iniciar sesion.

Puedo cambiar mi configuracion despues?

Si. Ejecuta jobhunter setup de nuevo. Tus datos anteriores aparecen como valores por defecto.

La IA inventa cosas en mi CV?

No. Solo reordena y destaca lo que ya esta en tu perfil. No agrega habilidades ni logros que no tengas.

Funciona para cualquier sector?

Si. Tecnologia, marketing, diseno, ventas, administracion, salud, educacion... cualquier sector con ofertas en LinkedIn.

Que hago si me sale un error?

La mayoria de errores son temporales. Espera unos minutos e intenta de nuevo. Si persiste, abre un issue en GitHub.
