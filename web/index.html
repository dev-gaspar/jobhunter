Caracteristicas

Todo lo que necesitas

Un sistema multi-agente de IA que automatiza cada paso de tu busqueda de empleo

Scraping inteligente

Escanea publicaciones de LinkedIn con Playwright, expande el texto completo, y extrae emails de reclutadores.

3 Agentes de IA

Agente filtrador, agente de CV, y agente de email. Cada uno especializado con Gemini AI para maxima precision.

CVs personalizados

Genera un CV unico por oferta, reescribiendo tu experiencia y habilidades para que encajen perfectamente.

Emails que suenan humanos

Emails cortos, directos, con logros concretos y numeros. Sin frases de plantilla.

Filtrado de duplicados

No envia el mismo cargo dos veces a la misma empresa. Diferentes cargos a la misma empresa si se permiten.

Cualquier sector

Funciona para tecnologia, marketing, ventas, diseno, administracion, salud, educacion y mas.

Como funciona

De cero a aplicaciones en minutos

Un comando para instalar, despues la IA hace el resto

1

Instalar

Un solo comando. Se instala globalmente como jobhunter en tu terminal.

2

Configurar

Sube tu CV, configura tu API key de Gemini (gratis), y define que tipo de empleo buscas.

3

Login en LinkedIn

Inicia sesion una vez. La sesion se guarda para futuras ejecuciones.

4

Ejecutar

El sistema escanea, analiza, genera CVs personalizados, escribe emails, y los envia automaticamente.
