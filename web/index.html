Todo lo que necesitas

Un sistema multi-agente de IA que automatiza cada paso de tu busqueda de empleo

🔍

Scraping inteligente

Escanea publicaciones de LinkedIn con Playwright, expande textos completos, y extrae emails de reclutadores automaticamente.

🤖

3 Agentes de IA

Agente filtrador, agente de CV, y agente de email. Cada uno especializado con Gemini AI para maxima precision.

📄

CV personalizado por oferta

Genera un CV unico para cada oferta, reescribiendo tu experiencia y habilidades para que encajen perfectamente.

📧

Emails que suenan humanos

Escribe emails cortos, directos, con logros concretos. Sin frases de plantilla ni texto generico.

🎯

Filtrado inteligente de duplicados

No envia la misma oferta dos veces a la misma empresa. Permite diferentes cargos en la misma empresa.

🌐

Para cualquier sector

Funciona para tecnologia, marketing, ventas, diseno, administracion, salud, educacion y mas.

Como funciona

De cero a aplicaciones enviadas en minutos

1

Configura tu perfil

Sube tu CV, configura tu API key de Gemini (gratis), y define que tipo de empleo buscas.

2

Login en LinkedIn

Inicia sesion una vez. La sesion se guarda para futuras ejecuciones.

3

Busqueda automatica

El sistema busca publicaciones relevantes, expande el texto completo, y extrae emails de contacto.

4

Analisis con IA

Un agente de IA analiza cada publicacion para determinar si es una oferta real y relevante para tu perfil.

5

CV + Email personalizados

Genera un CV y email unicos para cada oferta, optimizados para maximizar tus oportunidades.

6

Envio automatico

Se envia el email con el CV adjunto directamente al reclutador via Gmail.
